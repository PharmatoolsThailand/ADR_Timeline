Naranjo Algorithm

แบบประเมินโอกาสการเกิดอาการไม่พึงประสงค์จากยา (ADR Probability Scale)

คำถามพิจารณา (Question) ใช่ ไม่ใช่ ไม่ทราบ
≥ 9 ใช่แน่ (Definite)
5-8 น่าจะใช่ (Probable)
1-4 อาจจะใช่ (Possible)
≤ 0 น่าสงสัย (Doubtful)

คะแนนรวม: 0

ระดับความน่าจะเป็น: -

พัฒนาและออกแบบโดย: นายระพีพัฒน์ ศุภโกศล จบการศึกษาระดับปริญาตรี คณะเภสัชศาสตร์ มหาวิทยาลัยมหาสารคาม
